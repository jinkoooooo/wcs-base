## 1. 설명

본 문서는 역할 별 메뉴 권한 설정 지침을 담고 있습니다.

## 2. 역할 생성/수정/삭제

#### 2-1. 역할 생성

- 하단 `추가` 버튼 → `이름`, `Description` 작성 → `저장` 버튼

#### 2-2. 역할 수정

- 수정할 역할의 `이름`, `Description` 수정 → `저장` 버튼

#### 2-3. 역할 삭제

- 삭제할 역할의 `체크박스` 클릭 → `삭제` 버튼

## 3. 역할 별 메뉴 권한 부여

- 수정할 역할의 `역할 아이콘` 클릭 → 권한 수정할 상위 메뉴 클릭 → 하위메뉴의 CRUD 권한 체크 → `확인` 버튼
- > 로직: DB table permissions에서 해당 메뉴에 대한 기존 권한 전체 삭제 후, insert

## 4. 사용자 별 역할 부여

#### 4-1. 역할에 사용자 등록

- 수정할 역할의 `사용자 아이콘` 클릭 → `추가` 버튼 → `사용자` 탭에서 사용자 선택 → `저장` 버튼

#### 4-2. 역할에 사용자 삭제

- 삭제할 역할의 `사용자 아이콘` 클릭 → 삭제할 사용자 옆 `체크박스` 클릭 → `삭제` 버튼
